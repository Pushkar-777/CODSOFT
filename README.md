# QuizVerse AI

🚀 An interactive AI & Programming Quiz Application built using HTML, CSS and JavaScript.

## Features

- 30 AI & Programming Questions
- Real-Time Score Tracking
- Accuracy Percentage
- XP Reward System
- Modern Glassmorphism UI
- Responsive Design
- Instant Correct/Wrong Feedback

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Preview

QuizVerse AI helps users test their knowledge in:

- Programming Fundamentals
- Web Development
- Pseudocode Concepts
